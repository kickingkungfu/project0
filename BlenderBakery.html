Blender Bakery

There will be various components needed to put this project together.

The major hardware components needed will be a server (raspberry pi), desktop computers, and interrnet connections.

The major software components needed will be an operating system (Linux), application for managing the various users contributions to the animation environment, and server software for compiling the contributions from each of the contributors.

There have recently been some web applications that are similar to this idea, such as Mozilla's Hubs or Google's Game Builder on Steam.

My goal is to create an application that permits the full range of computer graphics imaging for the projects
